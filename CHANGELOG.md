# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-03-31

### Added
- Initial release of /skillify
- 4-phase workflow: context reconstruction, structured interview, SKILL.md generation, review & save
- agentskills.io standard compliance for cross-platform portability
- Auto-detection of project tooling (Node, Python, Go, Rust, Ruby, Make)
- Git artifact scanning for session context reconstruction
- Shell command blocks (`!`) for runtime project detection
- Install script with multi-platform detection (Claude Code, VS Code/Copilot, Gemini CLI)
- Example output demonstrating a generated PR review workflow skill
